## üöÄ Quick Start (5 Steps)

### Step 1: Generate Secrets
```bash
cd backend
bash setup-render.sh
# This generates: JWT_SECRET and .env.render file
```

### Step 2: Push to GitHub
```bash
git add .
git commit -m "Add Render deployment configuration"
git push origin main
```

### Step 3: Create Render Account
- Go to https://render.com
- Sign up with GitHub
- Create MySQL database service
- Create Web Service for backend

### Step 4: Configure Environment Variables
Copy from `.env.example` and set in Render Dashboard:
```
SPRING_DATASOURCE_URL=<from database>
SPRING_DATASOURCE_USERNAME=event_user
SPRING_DATASOURCE_PASSWORD=<from database>
JWT_SECRET=<from setup-render.sh output>
MAIL_USERNAME=<your-email@gmail.com>
MAIL_PASSWORD=<gmail-app-password>
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
```

### Step 5: Deploy and Test
```bash
# Render auto-deploys from GitHub
# Or click "Deploy" in dashboard
# Test with: curl https://your-service.onrender.com/api/events
```
